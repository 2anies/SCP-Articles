Часть SCP-001.

Объект №: SCP-001

Класс объекта: Эмбла

Особые условия содержания: SCP-001 содержится на территории Зоны 0 в сельском районе [УДАЛЕНО]. По периметру границы наблюдаемых эффектов объекта возведена ограда. Помимо команды службы безопасности Зоны 0, на территории Зоны постоянно должны находиться не менее пяти вооружённых охранников, которые обязаны пресекать любые попытки несанкционированного доступа. Рядом должна находиться полностью укомплектованная и работающая без перерывов физическая лаборатория, в которой занимаются изучением любых замеченных аномалий.

Маленькая металлическая мемориальная доска с надписью должна поддерживаться в хорошем состоянии. О любых повреждениях следует немедленно сообщать обслуживающему персоналу.

Описание: SCP-001 - это кольцевая гравиевая дорожка в лесистой местности. Если идти по ней против часовой стрелки, то тропа будет постоянно вести выше, даже после достижения точки отправления. Если идти по тропе по часовой стрелке, то число подъёмов совпадает с числом спусков, как и должно быть.

Для доступа к протоколу экспериментов SCP-001 необходим 5 уровень допуска.

Новые члены Совета Смотрителей должны ознакомиться с документом 001-O5.
