SCP-1186
(дата: ██.██.████)

Объект №: SCP-1186-RU

Класс объекта: Безопасный

Особые условия содержания: SCP-1186 должен храниться в сейфовой ячейке исследовательского сектора Зоны █. Исследования объекта разрешены сотрудникам с уровнем допуска 2 и выше, протокол предполагаемых экспериментов должен быть одобрен сотрудником с уровнем допуска не ниже 3. Все тесты проводятся на территории Зоны █ в стандартной испытательной камере, в радиусе 50 м от которой не должно находиться предметов, подходящих под определение "женская сумка" (см. Приложение 1186-C1-альфа). Для соблюдения чистоты экспериментов, все предметы, участвующие в тестировании, должны быть промаркированы радиоактивными изотопами.

Внимание! Сотрудники женского пола, находящиеся на любом сроке беременности, не допускаются к работе с объектом.

Описание: SCP-1186 представляет собой женскую кожаную сумку светло-коричневого цвета с массивной металлической застежкой, габаритами 32 x 47 x 21 см и массой 1,6 кг. Подобные модели были широко распространены в Европе в начале XIX века. На поверхности объекта имеется большое количество потертостей и других механических повреждений. Сумка изготовлена из натуральной кожи, предположительно оленьей, с подкладкой из атласной ткани, материалом застежки является сталь (содержание углерода – 1,9%).

Основное свойство объекта проявляется в случае, когда субъект (живой человек женского пола) опускает в него руку с целью извлечения предметов. При контакте с тканью подкладки, рука свободно проходит через нее, переходя, предположительно, в неевклидово пространство. При помощи SCP-1186 субъект способен извлекать предметы из так называемых "объектов-доноров". Объект-донор представляет собой обобщенное обозначение изделий, которые можно однозначно охарактеризовать определением "женская сумка" (см. Приложение 1186-C1-альфа). Область воздействия объекта составляет примерно 20 м и увеличивается, если в ее пределах находятся другие объекты-доноры (по-видимому, играющие роль ретрансляторов аномального эффекта) (см. Приложение 1186-D1).

Перечень предметов, которые возможно получить из других женских сумок, значительно варьируется для различных субъектов и основывается на подсознательном допущении о возможности извлечения из SCP-1186 конкретного предмета. Типичный набор включает: косметические средства, деньги, средства гигиены и контрацепции, различные мелкие вещи. В исключительных случаях возможно извлечение оружия и наркотиков.

При должной психологической подготовке, сотрудница Фонда смогла бы вытащить из SCP-1186 другой SCP-объект небольшого размера. При условии, что она допускает возможность его ношения в собственной сумке…
— д-р Сементьева

История обнаружения: Объект был обнаружен при расследовании серии преступлений, связанных с необъяснимыми исчезновениями материальных ценностей во время благотворительных вечеров для сбора пожертвований в период с ████ по ████ год в г. Москва. Для получения более подробной информации см. Приложение 1186-1

Приложение 1186-A-D/5: Протокол экспериментов

Предложение об использовании объекта для нужд мобильных оперативных групп находится на рассмотрении.
— O5-█
