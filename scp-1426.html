Объект №: SCP-1426

Класс объекта: Безопасный

Особые условия содержания: Для хранения SCP-1426-1 - SCP-1426-9 выделена сейфовая ячейка с кодовым замком в спецхран-исследовательской Зоне 76. Сотрудники могут использовать SCP-1426 в экспериментах только с письменного разрешения двух сотрудников с 3 уровнем допуска, а также одобрения директора Зоны.

Экземпляры SCP-1426 перед использованием должны быть тщательно стерилизованы с целью предотвращения загрязнения извлеченных образцов. Коронки следует применять только на дезинфицированных бетонных шлакоблоках, дабы уменьшить износ экземпляров.

Извлеченные образцы должны быть помещены в стерильные герметичные контейнеры и храниться в перчаточной камере в Исследовательской лаборатории 16 для исследования.

Описание: SCP-1426-1 - SCP-1426-9 - набор буровых алмазных коронок для взятия керна размерами 52x150 мм производства фирмы "██████ Electrical". При использовании SCP-1426 для бурения вещества извлеченный цилиндрический керн не соответствует по составу веществу, бурение которого проводилось. Вместо образца извлекается равное по объёму вещество из твердых тел Солнечной системы. Выбор небесного тела, из которого извлекается керн, определяется цветом используемой коронки. В некоторых случаях извлеченные с помощью SCP-1426 образцы проявляют свойства, нехарактерные для небесного тела, из которых были взяты (см. Приложение 1426-1). На боковине каждого экземпляра выгравировано название того небесного тела, из которого с его помощью берётся керн. Существует малая (0,35%) вероятность того, что образец не будет извлечен. Вместо этого внутри экземпляра, посредством которого производилось бурение, после окончания работ появляется свёрнутый в трубочку кусок бумаги. Содержание таких кусков варьируется в зависимости от используемой коронки, но в любом случае это будет написанная от руки копия фрагмента соответствующей части симфонии Густава Холста "Планеты". Если тело является спутником, то фрагмент будет относиться к части той планеты, спутником которой данное тело является. Если у данного небесного тела нет соответствующей части в симфонии, на бумаге будет отображена несуществующая часть, написанная в соответствии с последней существующей частью "Планет" ("Нептун").

Извлеченные с помощью SCP-1426 образцы сохраняют температуру космических тел, из которых они были извлечены. В одном эксперименте SCP-1426-2 был использован в сверлении шлакоблока, и извлеченный образец имел температуру, соответствующую имеющимся данным о температуре на поверхности Венеры. Ни выбранная для бурения коронка SCP-1426, ни буровой аппарат не проявляют признаков повреждения температурой извлеченных образцов, однако износ происходит так же, как и у обычных буровых коронок. Иногда полученные посредством SCP-1426 образцы имеют свойства, нехарактерные для небесного тела, из которого было произведено извлечение (см. Приложение 1426-1).

SCP-1426 были обнаружены в городе A██████, Англия, когда строитель █████ A█████ получил ожоги левой руки 3 степени при попытке извлечь из коронки часть кирпичной стены, которую он сверлил с помощью SCP-1426-2. Отчёт был обнаружен в ходе плановой разведывательной операции Фонда, после чего SCP-1426 был изъят и доставлен в спецхран-исследовательскую Зону 76. Исследование всех остальных буровых коронок в местном магазине ██████ Electrical не показало наличия у них аномальных свойств. Г-н A█████ и все свидетели происшествия были обработаны амнезиаком класса В и отпущены.

Приложение 1426-1: Имеющиеся экземпляры SCP-1426.
Цвет коронки: Серый
Тело, из которого извлекается образец: Меркурий
Примечание: Температура образцов зависит от температуры в Равнине Жары. Существует 12% вероятность того, что в образце будут присутствовать перья.
Обозначение: SCP-1426-1
Цвет коронки0: Оранжевый
Тело, из которого извлекается образец: Венера
Примечание: Температура образцов всегда в рамках 462-470°C. Иногда извлеченные образцы представляют собой расплавленную лаву, что указывает на возможную вулканическую активность.
Обозначение: SCP-1426-2
Цвет коронки0: Светло-голубой
Тело, из которого извлекается образец: Луна
Примечание: Состав образца идентичен таковому в Море Кризисов. Существует 15% вероятность того, что извлеченный образец будет иметь светло-синий оттенок и иметь запах сыра чеддер.
Обозначение: SCP-1426-3
Цвет коронки0: Красный
Тело, из которого извлекается образец: Марс
Примечание: Существует 5% вероятность того, что извлеченные образцы будут иметь очень высокую концентрацию водяного льда.
Обозначение: SCP-1426-4
Цвет коронки0: Бежевый
Тело, из которого извлекается образец: Ио
Примечание: Извлеченные образцы радиоактивны. Существует 85% вероятность того, что будет извлечена расплавленная сера.
Обозначение: SCP-1426-5
Цвет коронки0: Коричневый
Тело, из которого извлекается образец: Титан
Примечание: Существует 75% вероятность того, что будет извлечен жидкий этан. Кроме того, существует 0,65% вероятность того, что будет извлечена соленая вода.
Обозначение: SCP-1426-6
Цвет коронки0: Зеленый
Тело, из которого извлекается образец: Миранда
Примечание: Нет.
Обозначение: SCP-1426-7
Цвет коронки0: Темно-синий
Тело, из которого извлекается образец: Тритон
Примечание: Существует 25% вероятность того, что будет извлечен жидкий азот.
Обозначение: SCP-1426-8
Цвет коронки0: Розовый
Тело, из которого извлекается образец: Каждые 6 месяцев циклически переходит между Плутоном, Эридой, Макемаке и Кваваром
Примечание: Вне зависимости от того, из какой карликовой планеты извлечен образец, существует 15% вероятность того, что он будет содержать высокую концентрацию родия.
Обозначение: SCP-1426-9
