Объект №: SCP-3797

Класс объекта: Таумиэль

Уровень угрозы: Белый

Особые условия содержания: Поскольку SCP-3797 еще не существует, меры по содержанию не требуются.

Отдел временных аномалий займётся созданием SCP-3797. Для выполнения этой задачи его сотрудникам разрешено использовать другие SCP-объекты и временные аномалии с одобрения Совета O5.

После того, как создание SCP-3797 будет завершено, его необходимо использовать один раз, чтобы устранить SCP-3797-ARC в 07:42 по Всемирному координированному времени 13.06.2015. Затем объект должен быть демонтирован.

Описание: SCP-3797 будет представлять собой оружие, действующее по принципу обратной причинности, которое при активации должно будет уничтожить одну-единственную цель в прошлом. SCP-3797 будет использовано как минимум один раз в неопределенный день в будущем для устранения SCP-3797-ARC в 07:42 по Всемирному координированному времени 13.06.2015. Механизм работы SCP-3797 пока что неизвестен.

SCP-3797-ARC представляло собой существо высшего порядка, наделённое множеством аномальных способностей. Тем не менее, несмотря на свою практически божественную природу, SCP-3797-ARC не обладало даром предвидения. За дополнительной информацией, полученной во время спячки SCP-3797-ARC, обратитесь к Основному файлу SCP-3797-ARC.

13.06.2015 в 07:39 по Всемирному координированному времени SCP-3797-ARC пробудилось в соответствии с прогнозами и начало инициировать сценарий конца света класса CK. Для предотвращения этого события было дано разрешение на устранение объекта. МОГ Омега-9 («Клиника») и МОГ Омега-12 («Ахиллесовы пятки») попытались уничтожить SCP-3797-ARC, но все их усилия оказались напрасны из-за способностей SCP-3797-ARC к изменению реальности.

Эмблема Отдела временных аномалий.

13.06.2015 в 07:42 по Всемирному координированному времени над головой SCP-3797-ARC возникло несколько сотен эмблем Отдела временных аномалий. Пять секунд спустя плоть SCP-3797-ARC распалась на молекулы. На территории вокруг того места, где находилось SCP-3797-ARC, позже были обнаружены еще 72 эмблемы Отдела временных аномалий.

В это время датчики, используемые Отделом временных аномалий, зарегистрировали высокоэнергетический временной сдвиг в будущем.

Нынешнее объяснение грядущих событий гласит, что Отделу временных аномалий удалось нейтрализовать SCP-3797-ARC, поменяв местами во времени его смерть и причину смерти, в результате чего оно не смогло заранее узнать о намеченном устранении и использовать свои аномальные способности для предотвращения этого события.
