18.09.2011 внутри SCP-703 появился очередной SCP-703-1, представлявший собой список всех вещей, которые появлялись внутри SCP-703 с момента постановки на содержание. Данное обстоятельство лишний раз подтвердило, что SCP-703 является разумным неорганическим существом. В настоящий момент идет анализ документа.

Предмет: Алюминиевая банка.

Примечание: Весила 21.32 грамма.

Статус: Извлечена через 14.78 секунд после появления.

Предмет: Бумага красного цвета.

Примечание: На ней был нарисован круг.

Статус: Извлечена через 11.12 секунд после появления.

Предмет: Зеленая ручка.

Примечание: Заправлена красными чернилами.

Статус: Извлечена через 10.09 секунд после появления.

Предмет: Плюшевый тигр.

Примечание: По кличке "Полли".

Статус: Изъят через 9.00 секунд.

Предмет: Мой портрет.

Примечание: Приносит эстетическое удовольствие.

Статус: Изъят через 10.01 секунд.

Предмет: Белая краска.

Примечание: Может быть использована с целью реставрации.

Статус: Похищена через 9.89 секунд. Не была использована для реставрации.

Предмет: Три (3) экспансивных пули калибра .45

Примечание: Не удалось придать достаточное ускорение.

Статус: Похищены после столкновения с целью.

Предмет: Газообразное вещество.

Примечание: Не годится для дыхания.

Статус: Похищено посредством вдыхания через 20.6 секунд.

Предмет: Извинение.

Примечание: Составлено очень тщательно.

Статус: Извлечено через 45.78 секунд, без ответа.

Предмет: Сладкие штуковины.

Примечание: Так будет лучше

Статус: Не вышло

Предмет: Твой чемодан.

Примечание: Оставил в твоем офисе.

Статус: Извини за пятно.

Предмет: Стопка бумаг.

Примечание: Хочу помочь.

Статус: Помочь удалось.

Предмет: Замок более совершенной конструкции.

Примечание: Вас это беспокоит.

Статус: Извините.

Предмет: Один из черных ящиков.

Примечание: Извините, я сжульничал, хотел помочь перенести.

Статус: Прошел сквозь них. Сломан.
